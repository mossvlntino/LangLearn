# 🌐 LangLearn: AI Language Ecosystem

## 📖 Overview
A cross-platform (Web/Android) language learning companion powered by the Google Gemini API. This project simplifies multi-lingual education by combining a custom note-taking UI with a sophisticated chatbot capable of outputting native scripts, romanization, and translations simultaneously.

## 🛠️ Focus Areas
* **AI Integration:** Crafting reliable prompts for the Gemini engine.
* **Cross-Platform UX:** Maintaining feature parity between web and mobile interfaces.
* **Complex Data Rendering:** Handling and styling multiple global alphabets.
